Advanced Calendar PCF is a flexible and extensible calendar control built using the Power Apps Component Framework (PCF) to address common limitations of standard date pickers in Power Apps and Model-Driven Apps.

The control supports advanced date selection scenarios such as single date selection, date range selection, and is designed to be easily extended to support multiple date selection and event-based rendering.

Advanced Calendar PCF integrates seamlessly with Dataverse and can optionally bind to datasets, enabling it to display, select, and manage dates in complex business scenarios without relying on external services or AI-based features.

Key Features

Single date and date range selection

Extensible architecture for multi-date selection

Optional dataset binding for event-driven calendars

Responsive and Model-Driven App compatible layout

React-based implementation for maintainability

Microsoft-friendly design aligned with Power Platform best practices

Common Use Cases

Booking and reservation systems

Leave and absence management

Project scheduling and planning

SLA tracking and deadline management

Any scenario requiring advanced date selection logic

Design Goals

Go beyond standard date picker limitations

Maintain high performance and clean architecture

Avoid dependencies on Copilot or AI features

Enable gradual feature expansion without breaking changes
